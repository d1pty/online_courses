#  Online Courses (Spring Boot + React)

Веб-приложение для управления онлайн-курсами.
Backend реализован на **Spring Boot**, frontend — на **React**.

---

## Запуск backend (Spring Boot)

```bash
cd online_courses
mvn spring-boot:run
```

## Запуск frontend (React)

```bash
cd online_courses_frontend
npm install
npm start
```
